# img-newspaper

WebGLベースの画像フィルターで、カスタムHTML要素 `<img-newspaper>` を使用して新聞風のハーフトーン効果を適用します。

## デモ

[**ライブデモ**](~~https://code4fukui.github.io/img~~ *(unavailable)*-newspaper/)

インタラクティブなデモでは、画像をアップロードし、すべてのフィルターパラメーターをリアルタイムで調整できます。

## 機能

- **ハーフトーン効果:** クラシックな新聞風のモノクロハーフトーン（網点）効果を適用します。
- **高いカスタマイズ性:** ドットサイズ、網点角度、コントラスト、ガンマを調整できます。
- **アナログ感の再現:** インクの滲み、紙の質感、ビネット効果をシミュレートし、リアルな仕上がりを実現します。
- **カスタムカラー:** 紙とインクの色を自由に設定できます。
- **レスポンシブ対応:** アスペクト比を維持したまま、画像のサイズを要素に合わせて自動的に調整します。
- **簡単な組み込み:** シンプルなHTMLカスタム要素として、またはJavaScript API経由で使用できます。
- **エクスポート:** レンダリングされた画像をPNGとしてダウンロードできます（デモで利用可能な機能）。

## 要件

WebGLをサポートするモダンWebブラウザ。
